Component Features

  • Responsive Design: Automatically adapts to different screen sizes
  • Content Slots: Uses ng-content with named slots for flexible content
  • Customizable Dimensions: Configurable header height and sidebar width
  • Toggle Options: Show/hide left panel, right panel, and footer as needed
  • Accessibility: High contrast mode support and semantic HTML
  • Mobile Friendly: Collapsible sidebar on small screens

Usage Example

<aava-layout
  [headerHeight]="'100px'"
  [sidebarWidth]="'250px'"
  [footerText]="'© 2024 Company'"
  [showSidebar]="true"
  [showFooter]="true">

  <div slot="header">Header Content</div>
  <div slot="sidebar">Sidebar Content</div>
  <div>Main Content</div>
  <div slot="footer">Footer Content</div>
</aava-layout>

Footer Scrolling Behavior

Sticky Mode (Default): The footer remains fixed at the bottom of the viewport. Only the main content area scrolls. The footer is always visible.

Scrollable Mode: The footer scrolls naturally with the page content. It is not visible initially and only appears when you scroll to the bottom of the content. The entire body becomes scrollable instead of just the main content area.

Toggle between these modes using the "Sticky/Scrollable Footer" button in the controls above to see the difference in behavior.
