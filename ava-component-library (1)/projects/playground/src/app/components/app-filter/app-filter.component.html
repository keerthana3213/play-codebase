Filter Component

A versatile filter component that supports single and multiple selection, multiple filter groups, different selection types (checkbox/radio), and flexible customization. Perfect for product filtering, search refinement, and advanced data filtering scenarios.

Size Variants

Different sizes for various use cases and layouts

Size Options:

  • xs (Extra Small): Compact filters for limited space
  • sm (Small): Smaller interface elements
  • md (Medium): Default, balanced size
  • lg (Large): Larger text and controls
  • xl (Extra Large): Maximum visibility and touch-friendly

Selection Types

Different selection mechanisms for various filtering scenarios

Checkbox (Multiple)

Allow users to select multiple options

Radio Button (Single)

Allow users to select only one option

Display Only (None)

Show options without selection capability

Type Definitions

// Single filter option
interface AavaFilterOption {{'{'}
  id: string | number;
  label: string;
  value: any;
  selected?: boolean;
  icon?: string;
{{'}'}

// Filter group for multi-group mode
interface AavaFilterGroup {{'{'}
  title: string;
  selectionType: SelectionType;
  options: AavaFilterOption[];
{{'}'}

// Selection type options
type SelectionType = 'checkbox' | 'radio' | 'none';

// Filter size options
type FilterSize = 'xs' | 'sm' | 'md' | 'lg' | 'xl';
