🎛️ Drawer Behavior Control Properties

Control how your drawer behaves with these input properties. Each property has a specific purpose and use case.

showOverlay
boolean = true

Controls whether to show the dark background overlay behind the drawer.

[showOverlay]="true" - Standard modal behavior
[showOverlay]="false" - Non-modal sidebar
closeOnOverlayClick
boolean = true

Controls whether clicking the overlay closes the drawer.

[closeOnOverlayClick]="true" - Easy dismissal
[closeOnOverlayClick]="false" - Prevent accidents
closeOnEscape
boolean = true

Controls whether pressing the Escape key closes the drawer.

[closeOnEscape]="true" - Keyboard accessibility
[closeOnEscape]="false" - Critical workflows
showCloseButton
boolean = true

Controls whether to show the X close button in the header.

[showCloseButton]="true" - Clear exit option
[showCloseButton]="false" - Custom triggers
persistent
boolean = false

Makes the drawer "sticky" - cannot be closed by user actions.

[persistent]="false" - Normal behavior
[persistent]="true" - Force interaction
animate
boolean = true

Controls whether to use spring animation when opening/closing.

[animate]="true" - Smooth transitions
[animate]="false" - Performance mode
