Toast Component

A flexible and powerful toast notification system with multiple types, positions, and customization options. Built with service pattern architecture for easy integration and management. Hover over toasts with auto-dismiss to pause the timer!

ToastService Methods

Method Parameters Return Type Description
success(config?) SuccessToastConfig Promise<ToastResult> Show a success toast notification
error(config?) ErrorToastConfig Promise<ToastResult> Show an error toast notification
warning(config?) WarningToastConfig Promise<ToastResult> Show a warning toast notification
info(config?) InfoToastConfig Promise<ToastResult> Show an info toast notification
custom(config?) CustomToastConfig Promise<ToastResult> Show a custom toast notification
setPosition(position) ToastPosition void Set the global toast position
dismissAll() - void Dismiss all active toasts

Configuration Options

Property Type Default Description
title string undefined Toast title text
message string undefined Toast message text
duration number 4000 Auto-dismiss duration in milliseconds
position ToastPosition 'top-right' Toast position on screen
showCloseButton boolean true Show close button
showProgress boolean true Show progress bar
icon string auto Custom icon name (Lucide icons)
iconColor string auto Custom icon color
customWidth string '400px' Custom toast width (custom type only)
customHeight string 'auto' Custom toast height (custom type only)
