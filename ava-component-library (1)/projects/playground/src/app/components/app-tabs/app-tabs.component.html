Custom Tab Row (Pill Shape & Gradient Background)

The tab row and its background can be fully customized using the tabRowWrapperStyles and tabRowBackgroundStyles props. This example shows a pill-shaped row with a gradient background.

Custom Tab Row (Colored & Rounded)

You can also apply a colored background and rounded corners to the tab row for a different look.

Navigation-Only Tabs

Use showContentPanels="false" for pure navigation tabs without content panels. Perfect for navigation bars, menus, or when content is handled elsewhere.

Event Handling

Open browser console to see event outputs when interacting with tabs.

Available Events:

  • tabChange: Emitted when a tab is selected
  • tabClose: Emitted when a closeable tab is closed
  • dropdownToggle: Emitted when dropdown is opened/closed

Component Properties

Property Type Default Description
tabs TabItem[] [] Array of tab items to display
activeTabId string '' ID of the currently active tab
variant 'default' | 'button' | 'iconOnlySquare' | 'iconOnlyCircle' 'default' Visual style variant
bordered boolean false border enable and disabling
size 'xs' | 'sm' | 'md' | 'lg' | 'xl' 'md' Size of the tabs
disabled boolean false Disable all tabs
scrollable boolean false Enable horizontal scrolling
showDropdown boolean false Show dropdown for hidden tabs
maxVisibleTabs number 5 Maximum visible tabs before dropdown
allowTabClose boolean false Allow closing tabs with close button
centeredTabs boolean false Center align the tab list
fullWidth boolean false Make tabs fill full width
