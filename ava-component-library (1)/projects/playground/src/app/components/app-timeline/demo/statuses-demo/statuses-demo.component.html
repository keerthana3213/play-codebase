Timeline Status Types Demo

Visual status indicators for timeline events with different states. Each status provides visual feedback about the current state of events.

Status-Based Timeline

Timeline showing different status types with color-coded indicators.

Status Types Guide

Completed

Events that have been finished successfully. Usually shown with green styling.

Project Planning - Finished

Current

Events that are currently in progress or active. Usually shown with blue styling.

Design Phase - In Progress

Pending

Events that are scheduled but not yet started. Usually shown with orange styling.

Development - Scheduled

Unreached

Future events that are not yet accessible. Usually shown with gray styling.

Testing & QA - Future
