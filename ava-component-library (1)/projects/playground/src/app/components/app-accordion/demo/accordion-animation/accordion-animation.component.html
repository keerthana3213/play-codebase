Animation Best Practices

Guidelines for using accordion animations effectively in your applications.

✅ When to Use Animations

  • Most user interfaces and web applications
  • When you want to provide smooth visual feedback
  • For content that benefits from gradual revelation
  • When building modern, polished user experiences

❌ When to Disable Animations

  • Performance-critical applications
  • When users prefer reduced motion
  • For frequently toggled content
  • When building minimalist interfaces
