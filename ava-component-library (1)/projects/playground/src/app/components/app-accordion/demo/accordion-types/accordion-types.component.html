Use Cases

When to use each accordion type based on your content and design needs.

Default Type - Best For:

  • Simple content organization
  • Traditional accordion patterns
  • When you want consistent icon behavior
  • General purpose expandable content

Title Icon Type - Best For:

  • Content categorization (folders, categories)
  • When you need static visual indicators
  • Settings panels and configuration sections
  • Navigation menus with icons
